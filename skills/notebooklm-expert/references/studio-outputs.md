# Studio Outputs Detail

Access Studio via the right panel (toggle "Studio" if not visible).

## 1. Audio Overviews (AI Podcast)

**Default**: General overview (fine for first listen, not targeted)

**Better**: Click pencil icon before generating
- **Format options**: Deep Dive, Debate, Critique, Brief
- **Add focus instruction**:
  ```
  Focus on whether our pricing model is competitive enough based on market data.
  Challenge any assumptions in the beta user feedback.
  ```

**Result**: Two hosts argue different positions using your data

**Interactive Mode**: Click "Join" to ask questions mid-conversation
- Hosts pause, answer using notebook info, resume discussion

**Use case**: Listen during commute; if hosts poke holes in your pricing argument, fix before presenting

## 2. Reports

**Preset Formats** (top of panel):
- Briefing Doc
- Strategic Playbook
- Operational Framework

**Suggested Formats** (auto-generated based on your content):
- NotebookLM analyzes what's in your notebook
- Recommends formats matching your sources

**Custom Reports**: "Create your own" -> Describe exactly what you need

**Example Output**:
- Multi-page executive summary
- Findings from every source
- Execution checklist
- Generated in ~90 seconds (vs. 1.5 hours manually)

## 3. Data Tables (Most Valuable for Business Use)

NotebookLM pulls data from across sources -> Organizes into structured tables -> Export to Google Sheets

**Example Prompt**:
```
Create a comparison table of all three competitors with columns for:
- Pricing tiers
- Free trial length
- Onboarding support
- Biggest weakness mentioned in my sources
```

**Result**: Clean table exported, edited, sent in ~1 minute
