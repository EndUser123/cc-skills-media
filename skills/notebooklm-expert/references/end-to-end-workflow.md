# End-to-End Workflow Example

**Scenario**: Product launch strategy for stakeholder meeting on Thursday

## Day 1: Setup

1. Create notebook: "B2B SaaS Product Launch"
2. Add sources:
   - Market sizing report (target segment)
   - 3 competitor landing pages (reviewed manually)
   - Internal survey data (beta users)
   - Product lead firsthand account
3. Deep Research:
   ```
   B2B SaaS launch strategies. Focus: industry reports, case studies.
   Exclude: blog posts, listicles.
   ```
4. Review each suggested source -> Import if credible
5. Contradiction check:
   ```
   Flag any conflicting data across market research and sizing reports
   ```

## Day 1: Configuration

6. Configure Notebook:
   - Custom role: "Product marketing strategist preparing B2B SaaS launch. Prioritize competitive positioning, messaging clarity, pricing sensitivity, GTM sequencing."
   - Response length: Longer

## Day 2: Analysis

7. **Analyze**:
   ```
   What are the strongest positioning angles for our launch based on competitor weaknesses and beta feedback?
   ```
8. **Challenge**:
   ```
   What are the weakest assumptions in this positioning analysis? Which claims lack supporting evidence?
   ```
9. **Gap**:
   ```
   What's missing from my sources to complete this positioning analysis?
   ```
10. Save best answers -> Convert to Source
11. Upload missing sources identified in Gap step

## Day 3: Deliverables

12. Studio -> Audio Overview:
    - Format: Deep Dive
    - Focus: "Pricing competitiveness based on market data. Challenge beta user assumptions."
    - Listen during commute -> Fix pricing argument if hosts find holes
13. Studio -> Reports:
    - Generate Briefing Doc with execution checklist
14. Studio -> Data Tables:
    - Competitor comparison table (pricing, trials, onboarding, weaknesses)
    - Export to Sheets

## Day 4: Meeting

15. Walk in with:
    - Formatted briefing doc (looks like full day's work)
    - Execution checklist
    - Competitor comparison table
    - Pre-empted stakeholder questions (from Challenge step)
    - Confidence all claims are citable to sources
